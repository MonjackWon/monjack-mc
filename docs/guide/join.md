# 如何加入服务器

## 下载PCL2

进入官网（[https://pcl2.org/](https://pcl2.org/)）下载

也可以选择进入[爱发电](https://afdian.com/p/0164034c016c11ebafcb52540025c377)下载快照版支持作者

## 导入整合包

1. 从[蓝奏云](https://wwatx.lanzouu.com/i6Qq63g6ewkh)网盘下载
2. 打开PCL，点击*版本选择，*再点击*导入整合包*选择刚刚下载到本地的压缩包文件（不要解压！！）
3. 等待资源下载完成

## 加入服务器

1. 回到PCL启动器首页
2. 选择正版登录或者使用离线游玩（需要输入你的用户名）
3. 等待客户端启动
4. 启动完成后，点击多人游戏
5. 选中服务器双击进入
6. 现在可以愉快地游玩了
